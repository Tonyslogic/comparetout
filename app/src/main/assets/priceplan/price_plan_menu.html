Help: Price Plan Menu

Menu item Description
Edit Enables edit mode for the currently visible price plan.
Save Saves the currently active price plan to the database. Only valid plans can be saved to the database.
Info Displays a short message indicating why the price plan has failed validation. Only valid plans can be saved to the database. Only visible if validation fails.
Add a day rate Adds a new day rate tab (last position).
Delete a day rate Removes the currently visible day rate tab.
Share a price plan Shares the currently active price plan as a json document. Only valid plans can be shared.
