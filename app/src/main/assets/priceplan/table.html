Help: Costs detail table

The cost defines a contract with a supplier. This is also referred to as a price plan

Use this tab to record the non-rate aspects of a supplier plan:

  • Supplier. The name of the supplier
  • Plan. The name of the plan
  • Feed in rate. What the supplier pays for exported electricity
  • Standing charges. Annual cost of the contract without using any electricity
  • Sign up bonus. Some suppliers provide a monetary enticement to switch
  • Last update. Plans change all the time, this helps track the validity of calculations
  • Reference. Free text. Can be used to record the source of the data
  • Deemed export. Indicates how the total export value should be calculated
