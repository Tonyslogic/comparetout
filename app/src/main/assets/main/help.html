Help: App overview

The goal of this app is to create personalised comparisons.

There are two components to electricity comparisons:

  • The consumer side (usage)
  • The supplier side (cost)

Comparisons can be created for a single supplier with multiple variation in usage. They may also be made for a single usage across multiple supplier plans.

There are three tabs presented:

Tab Purpose
USAGE(S) Usage what-if scenario management/navigation and add/remove from comparison
USAGE(S) Supplier plan (cost) management/navigation and add/remove from comparison
COMPARE Show the results of the cost calculation, including breakdowns

At least one usage and one cost must be selected for anything to appear in the comparison tab.

There are several ways to get usages and costs:

  • Load from file (menu)
  • Download from URL (menu)
  • Copy and existing entry
  • Create a new entry from scratch (button, bottom right)

Additionally, sharing is possible for usages, costs and comparisons. For usages and costs, a json string is shared. For comparisons a csv of all the usages and costs is generated. These can be saved or shared based on the capabilities of you device.

Clicking on the table entry will:

Tab Item Result
USAGE(S) Name Launch the usage what-if scenario viewer/editor
COSTS Name Launch the usage supplier plan viewer/editor
COMPARE Name Display a breakdown of the purchased electricity by rate in a pie chart
USAGE(S) Delete Delete the entry after confirmation.
COSTS Delete Delete the entry after confirmation
USAGE(S) Copy Create a new entry with an '_copy' suffix
COSTS Copy Create a new entry with an '_copy' suffix

On the filter tab, it is possible to filter the fields shown, and sort based on any of the fields.
