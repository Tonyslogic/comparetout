Help: Octopus Energy importer

Octopus Energy provides API access to your half-hourly smart meter history (import, and export where you have an export meter point), plus the tariffs you are on. You need your account number (A-XXXXXXXX) and an API key from the Octopus dashboard (Account → API access).

When your data is fetched, the app also creates supplier plans from the Octopus tariffs available in your region, and marks your current tariff as the favourite plan (unless you already chose one). Dynamic tariffs (Agile/Tracker) cannot be represented as repeating plans and are skipped.

No API key? Download your consumption CSV from the Octopus dashboard and import it with the folder icon above (import readings only).

There are three tabs presented:

Tab Purpose Details
Octopus Overview Credentials, fetch, remove and cost range Overview
Data graphs Visualise downloaded data in multiple different ways Graphs
Generate usage Create a new usage what-if scenario from a subset of the downloaded data Generate

Long clicking on a tab title will provide more detailed help.
