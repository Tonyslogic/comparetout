Help: Octopus overview tab

Set credentials: enter your Octopus account number (A-XXXXXXXX) and API key. The key is generated on the Octopus dashboard under Account → API access. Credentials are stored encrypted on the device.

Select system: valid credentials auto-discover your electricity meter points (MPANs). Pick the one to work with — an export meter point on the same property is fetched automatically alongside it.

Fetch data: choose a start date; the fetch walks forward to today and a daily worker keeps it up to date overnight. Import fills the buy series, export fills feed.

Remove data deletes stored readings for the selected system — all of them, or just a selected date range.

Cost data prices the selected date range against your supplier plans. Note that as meter data, gross consumption behind solar panels is not visible: self-consumed generation never crosses the meter, so the load seen here understates a solar home's true usage (the same caveat as ESBN).
