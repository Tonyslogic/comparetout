The AlphaESS historical data may be used to generate a usage what-if scenario. There is a lot of information in the data and this tab provides options on which data to include in the usage.

There are two actions:

Action Purpose Details
PICK DATE RANGE Pick a single date (the start of a year) The usage needs a full years data to work. The date must be at least one year before the last available date. Invalid selections are indicated below the action buttons. Valid ranges are also indicated below the action buttons
GENERATE USAGE Generate a new Usage from the inverter data, using the selected date range and the options specified.

This action is disabled until there is valid input for generation.

Generation can take a long time. Generation runs in the background. Progress is visible via notificaitons, and status text below the action buttons.

There are six options for inclusion in a generate action:

Option Purpose Details
Generate load profile Create a load profile in the usage.

This option is disabled until there is a valid date range.

Use the 5 minute load data to create a load profile. This is mandatory for generation. Once selected, other options are enabled.
Generate inverter Create an inverter in the usage.

This option is disabled until there is a load profile.

The raw data is queried to get the vital statistics for the inverter.

When selected, you must provide the number of MPPT connections
Generate panel Create panel string in the usage.

This option is disabled until there is an inverter.

AlphaESS data is used to calculate the kWp of each panel (assumed to be the same).

When selected, you must provide the number of panels on each MPPT connection
Generate panel data Create panel data for each string in the usage.

This option is disabled until there are panel(s) defined.

AlphaESS data is used to provide 5 minute PV generation data to the usage. The PV for each 5 minute interval is distributed across all strings in proportion to the number of panels they host.
Generate battery Create battery settings in the usage.

This option is disabled until there is an inverter.

AlphaESS data is used to calculate the battery characteristics and charge model.
Generate battery schedule Create battery schedules in the usage.

This option is disabled until there is a battery.

AlphaESS data is searched for charging patterns. This may result in a lot of schedules. Extra schedules may be deleted, and incomplete schedules fixed in the usage generated.
