HomeAssistant provides access to historical energy data that may be used to seed a what-if usage scenario.

There are four tabs presented:

Tab Purpose Details
HomeAssistant Overview Credentials, fetch, remove and cost range Overview
Data graphs Visualise downloaded data in multiple different ways Graphs
Key stats Some fun facts and comparisons Stats
Generate usage Create a new usage what-if scenario from a subset of the downloaded data Generate

Long clicking on a tab title will provide more detailed help.
