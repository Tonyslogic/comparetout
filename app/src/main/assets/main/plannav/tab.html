Help: Costs tab

The cost defines a contract with a supplier. This is also referred to as a price plan

Use this tab to manage usage costs. You may:

  • Add a new cost (button, bottom right)
  • Delete a not needed cost
  • Select costs for inclusion in comparisons
  • Launch the cost editor/viewer
  • Copy an existing cost price plan
  • Load cost price plans from a file (menu)
  • Download cost price plans from a URL (menu)
  • Save the cost price plans to a file
