Help: Usage tab

The usage defines a what-if scenario for how electricity is used, stored and generated in the house.

Use this tab to manage usage what-if scenarios. You may:

  • Add a new usage what-if scenario (button, bottom right)
  • Delete a not needed what-if scenario
  • Select the usage what-if scenario for inclusion in comparisons
  • Launch the editor/viewer
  • Copy an existing usage what-if scenario
  • Load usage what-if scenarios from a file (menu)
  • Download usage what-if scenarios from a URL (menu)
  • Save the usage what-if scenarios to a file
