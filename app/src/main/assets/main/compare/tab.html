Help: Compare tab

The comparison tab shows a summary table of the costs for each usage what-if scenario.

Use this tab to see the results of selected cost calculations. You may:

  • Filter columns displayed
  • Specify the column to be sorted
  • See a breakdown of overall cost by rate (pie chart)
  • Save the comparison data to a file

Using this tab you can compare a single usage what-if scenario across a list of different supplier price plans (costs). This is quite useful once you are happy that you have selected the usage that best fits your goals.

Alternatively, usage what-if scenarios can be explored with a single supplier price plan (cost) to help determine either the best usage of your existing system, or to estimate the potential savings of a new investment.
