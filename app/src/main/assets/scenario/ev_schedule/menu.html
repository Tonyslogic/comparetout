Help: EV schedule menu

In edit mode, EV schedules can be added (button in bottom right), and deleted (menu)

The EV menu provides options to:

  • Edit. Enter edit mode.
  • Save. Write changes to the database. Exit edit mode
  • Share. Export, share backup these schedules.
  • Import. Replace the content of this schedule with an imported one.
  • Copy. Copy a schedule from another usage what-if scenario.
  • Link. Link this schedule to another usage what-if scenario.
  • Delete. Delete this schedule from the usage what-if scenario.

Copying an EV schedule means that a new entry is made in the database and the details of the source copied. It has the same effect as creating the same EV schedule twice.

Linking an EV schedule means that a change to the schedule will impact all linked usage what-if scenarios. When EV schedules are linked, a button will appear to the left of the scheduled time. Clicking that button will briefly display a list of linked usage what-if scenarios.
