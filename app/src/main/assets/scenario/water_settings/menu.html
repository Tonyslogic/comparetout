Help: Water settings menu

The water settings menu provides options to:

  • Edit. Enter edit mode.
  • Save. Write changes to the database. Exit edit mode
  • Share. Export, share backup these inverters.
  • Import. Replace the content of this inverter with an imported one.
  • Copy. Copy an inverter from another usage what-if scenario.
  • Link. Link this inverter to another usage what-if scenario.

Copying a water settings means that a new entry is made in the database and the details of the source copied. It has the same effect as creating the same water settings twice.

Linking a water settings means that a change to the settings will impact all linked usage what-if scenarios. When water settings are linked, a button will appear in the bottom right. Clicking that button will briefly display a list of linked usage what-if scenarios.
