Help: Load profile menu

The load profile menu provides options to:

  • Edit. Enter edit mode.
  • Save. Write changes to the database. Exit edit mode
  • Share. Export, share backup this load profile.
  • Import. Replace the content of this load profile with an imported one.
  • Copy. Copy a load profile from another usage what-if scenario.
  • Link. Link this load profile to another usage what-if scenario.

Copying load profile means that a new entry is made in the database and the details of the source copied. It has the same effect as creating the same load profile twice.

Linking load profile means that a change to the load profile will impact all linked usage what-if scenarios. When load profiles are linked, a button will appear in the bottom right. Clicking that button will briefly display a list of linked usage what-if scenarios.
