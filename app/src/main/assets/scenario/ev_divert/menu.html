Help: EV divert menu

In edit mode, EV divert schedules can be added (button in bottom right), and deleted (menu)

The EV menu provides options to:

  • Edit. Enter edit mode.
  • Save. Write changes to the database. Exit edit mode
  • Share. Export, share backup these divert schedules.
  • Import. Replace the content of this divert schedule with an imported one.
  • Copy. Copy a divert schedule from another usage what-if scenario.
  • Link. Link this divert schedule to another usage what-if scenario.
  • Delete. Delete this divert schedule from the usage what-if scenario.

Copying an EV divert schedule means that a new entry is made in the database and the details of the source copied. It has the same effect as creating the same EV divert schedule twice.

Linking an EV divert schedule means that a change to the schedule will impact all linked usage what-if scenarios. When EV divert schedules are linked, a button will appear to the left of the scheduled time. Clicking that button will briefly display a list of linked usage what-if scenarios.
