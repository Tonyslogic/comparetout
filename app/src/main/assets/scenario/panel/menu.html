Help: Panel menu

The panel menu provides options to:

  • Edit. Enter edit mode.
  • Save. Write changes to the database. Exit edit mode
  • Share. Export, share backup these panels.
  • Import. Replace the content of this panel with an imported one.
  • Copy. Copy a panel from another usage what-if scenario.
  • Link. Link this panel to another usage what-if scenario.
  • Delete. Delete this panel from the usage what-if scenario.

Copying a panel means that a new entry is made in the database and the details of the source copied. It has the same effect as creating the same panel twice.

Linking a panel means that a change to the panel will impact all linked usage what-if scenarios. When panels are linked, a button will appear in the bottom left. Clicking that button will briefly display a list of linked usage what-if scenarios.
