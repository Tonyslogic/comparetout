Help: Inverter menu

The inverter menu provides options to:

  • Edit. Enter edit mode.
  • Save. Write changes to the database. Exit edit mode
  • Share. Export, share backup these inverters.
  • Import. Replace the content of this inverter with an imported one.
  • Copy. Copy an inverter from another usage what-if scenario.
  • Link. Link this inverter to another usage what-if scenario.
  • Delete. Delete this inverter from teh usage what-if scenario.

Copying an inverter means that a new entry is made in the database and the details of the source copied. It has the same effect as creating the same load profile twice.

Linking an inverter means that a change to the inverter will impact all linked usage what-if scenarios. When inverters are linked, a button will appear in the bottom right. Clicking that button will briefly display a list of linked usage what-if scenarios.
