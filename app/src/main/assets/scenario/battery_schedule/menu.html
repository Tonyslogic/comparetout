Help: Battery charging menu

The battery settings menu provides options to:

  • Edit. Enter edit mode.
  • Save. Write changes to the database. Exit edit mode
  • Share. Export, share backup these batteries.
  • Import. Replace the content of this battery settings with an imported one.
  • Copy. Copy a battery settings from another usage what-if scenario.
  • Link. Link this battery settings to another usage what-if scenario.
  • Delete. Delete this battery settings from teh usage what-if scenario.

Copying a battery settings means that a new entry is made in the database and the details of the source copied. It has the same effect as creating the same battery settings twice.

Linking a battery settings means that a change to the settings will impact all linked usage what-if scenarios. When battery settings are linked, a button will appear in the bottom right. Clicking that button will briefly display a list of linked usage what-if scenarios.
